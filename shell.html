Tips:

When composing complex formulas, switch to float data type first to avoid confusion.

Use Ctrl key when clicking the + or - buttons for a bigger increment or decrement step.

Select any number in the desired textfield and write e.g. *2 to multiply the value by two.
Available operators: *, /, +, -.

Right click on the plot to see more options.
When the auto fit option is disabled, double click on the plot to auto fit the view.

Use negative amplitude values to mirror vertically the wave shape.

Available formula utils:

Variables:
t - current sample index (from 0 to T)
T - number of samples per one period
rand - fresh random number (from -1.0 to 1.0)

Constants:
pi, e

Functions:
abs(x), acos(x), asin(x), atan(x), atan2(x,y), ceil(x), cos(x),
cosh(x), exp(x), fac(x), floor(x), ln(x), log(x), log10(x), ncr(x,y),
npr(x,y), pow(x,y), sin(x), sinh(x), sqrt(x), tan(x), tanh(x).

Operators:
+, -, *, /, ^, %, (, ), ,
